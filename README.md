# NammaMela – Android App Development using GenAI

## Overview
NammaMela is an Android application developed to support traditional village drama troupes and mela events. The app helps users discover drama performances, view cast details, and book tickets easily.

## Features
- Browse village mela events
- View drama troupe information
- Check cast details
- Ticket booking support
- User-friendly Android UI
- Built using Kotlin in Android Studio

## Technologies Used
- Kotlin
- Android Studio
- RecyclerView
- XML Layouts

## Objective
The application digitally supports traditional cultural events and improves accessibility for audiences.
